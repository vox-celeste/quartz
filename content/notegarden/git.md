resources for learning git:

* [Learn Git Branching](https://learngitbranching.js.org/) – an interactive git visualization tool to educate and challenge
* [How Git Works](https://wizardzines.com/zines/git/) – zine explaining git’s core concepts (commits! branches! merging! remotes!) with minimal jargon and a focus on the actual problems that can ruin your day
* [Oh shit, git!](https://wizardzines.com/zines/oh-shit-git/) – zine explaining git fundamentals and how to fix a lot of common git mistakes
